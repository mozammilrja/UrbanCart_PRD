# UrbanCart PRD

Product Requirements Document for **UrbanCart** - a next-generation Indian streetwear e-commerce platform.

## Overview

UrbanCart is a digitally-native, culturally-grounded Indian streetwear platform targeting creative professionals aged 22-35 in Indian metro cities. The platform combines designer-led narrative identity, multi-brand cultural retail, and high-performance e-commerce architecture.

## Business Model

- **70%** Primary brand revenue
- **20%** Curated collaborations
- **10%** Community-driven commerce (memberships, events, exclusives)

## MVP Scope

- Full e-commerce platform with product catalog
- User authentication
- Cart/checkout with Indian payment methods (Razorpay/UPI)
- Order management
- Admin panel for content and order management
- Built as a Turborepo monorepo

## 5-Year Vision

- ₹200+ Cr ARR
- 1M+ customers
- 5 flagship stores ("Chapters")
- International presence
- Curated marketplace ecosystem
